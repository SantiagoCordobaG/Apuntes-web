# proyectexample

Aplicación web de gestión de documentos con autenticación de usuarios, subida de archivos, etiquetado automático y valoración. El proyecto combina un frontend en Vue 3 con un backend Node.js/Express para gestionar usuarios, documentos y servicios asociados.

## Qué hace este proyecto
- Registro e inicio de sesión de usuarios.
- Subida y gestión de documentos.
- Etiquetado automático de contenido.
- Valoración de documentos y perfil de usuarios.

## Estructura del proyecto
- `src/`: frontend de Vue 3.
- `backend/`: servidor Express y APIs.
- `public/`: archivos estáticos.
- `src/components/`: componentes de la interfaz.
- `src/stores/`: gestión de estado con Pinia.
- `src/router/`: rutas de la aplicación.
